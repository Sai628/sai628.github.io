Markdown 语法概要

Posted on 2016-01-28

字体

1. 斜体
在字符串的前后添加下划线 _.

This is italics

2. 粗体
在字符串的前后添加两个星号 **.

This is bold

当然,如果想斜体同时粗体的话, 可以像下面这样:

This is italics and bold

字号

在某段字符串的最前面添加n个 # 号(n为1-6), 可以使该行变为 header 效果. 注意在 # 与文本之间添加一个空格.

效果如下:

This is header one

This is header two

This is header three

This is header four

This is header five
This is header six

链接

[]后加上() []中为待添加链接的字符串, ()为链接对应的URl
